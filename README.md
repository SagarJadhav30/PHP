## Features
- Basic PHP web page (index.php)
- Custom styling (style.css)

## Getting Started

### Prerequisites
- [XAMPP](https://www.apachefriends.org/) or any local web server with PHP support

### Installation & Usage
1. **Clone or Download the Repository**
   - Click "Code" > "Download ZIP" or use:
     ```bash
     git clone https://github.com/your-username/Sj.git
     ```
2. **Move the Project Folder**
   - Place the `Sj` folder inside your web server's root directory (e.g., `htdocs` for XAMPP).
3. **Start the Web Server**
   - Open XAMPP and start the Apache server.
4. **Access the Project**
   - Open your browser and go to:
     ```
     http://localhost/Sj/
     ```
